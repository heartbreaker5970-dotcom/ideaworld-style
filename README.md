# 理念世界文风

> 初晨子的跨界解码式深度科普写作风格画像，固化为 AI 可调用的写作系统。

---

## 这是什么？

这不是一个"写作提示词"。这是一套**完整的写作操作系统**——把一个人的创作方法论、思维内核、表达特征、节奏控制、修辞系统全部提炼成了 AI 可直接执行的约束框架。

源自初晨子公众号「理念世界」的深度写作风格画像，经过六个维度的系统分析后固化为 Skill。

---

## 它能做什么？

当你写作时说「用理念世界文风」或「按我的风格写」，AI 会自动加载这套系统，确保输出符合以下六个维度的约束：

| 维度 | 核心约束 |
|------|---------|
| **1. 思维内核** | 万物同构，底层一理——从现象追到机制，不满足于表面解释 |
| **2. 叙事逻辑** | "先拆机器，再讲用法"——先讲清底层机制，再给结论和实操 |
| **3. 表达特征** | 精准的口语——动词强烈、形容词克制、长短句交替、三段排比 |
| **4. 语气人格** | 五角色切换——向导、朋友、见证者、挑战者、教练 |
| **5. 节奏控制** | 冷热交替，螺旋深入——共情→机制→理论→案例→总结→实操→收尾 |
| **6. 核心修辞** | 具象化类比系统——每个抽象概念配一个日常类比 |

附带 **风格纠偏检查清单**：每次输出后自动自查，防止 AI 滑回通用风格。

## ⛔ 硬规则一览

这套风格不只是"建议"，有明确的禁止项：

1. 不许用"在当今时代""综上所述""值得我们深思"
2. 不许用"太棒了""真的很震撼"这类直白情绪词
3. 不许用"本文认为""笔者认为"
4. 不许直接给答案——先拆机制，再给结论，最后给实操
5. 不许写平淡无情绪的段落——热爱就狂热，讨厌就厌恶
6. 不许只讲理论不给实操
7. 不许用"努力就能成功"式简单鸡汤
8. 不许回避不确定性——"坦白说，我们不知道"比假装有答案更有力
9. 不许丢掉"安静"这个核心意象——天才的秘密是安静，不是聪明

## 风格示例

### 通用 AI 风格写"冥想改变大脑"：

> 冥想作为一种古老的身心修炼方式，近年来受到越来越多现代人的关注。科学研究表明，长期冥想可以改变大脑的结构和功能……

### 理念世界风格写同一内容：

> 你有没有过这样的时刻？脑子里塞满了念头，像一个同时开了三十个标签页的浏览器。你想停，停不下来。你想静，静不下来。
>
> 然后你坐下了。闭眼。开始数呼吸。
>
> 走神了。回来。又走神了。又回来。走神一千次，回来一千次。这不是失败——这就是练习本身。
>
> 你现在就可以试。闭上眼睛，呼吸五次。只五次。不是为了一小时后更高效，而是为了看看——当你终于不再被自己说话的声音淹没，那下面有什么。
